# Credit Card Fraud Detection

## Description

Credit Card Fraud Detection is a machine learning project designed to identify fraudulent credit card transactions by analyzing transaction patterns and customer behavior. The system uses historical transaction data to train classification models that distinguish between legitimate and fraudulent activities.

The project includes data preprocessing, feature engineering, model training, evaluation, and prediction. Various machine learning algorithms such as Logistic Regression, Decision Tree, Random Forest, and XGBoost can be used to improve detection accuracy. Performance is evaluated using metrics such as precision, recall, F1-score, and ROC-AUC, which are especially important for handling imbalanced datasets.

## Features

* Detects fraudulent credit card transactions.
* Handles highly imbalanced datasets using preprocessing techniques.
* Performs data cleaning and feature scaling.
* Trains and compares multiple machine learning models.
* Evaluates model performance using accuracy, precision, recall, F1-score, and ROC-AUC.
* Predicts whether a transaction is legitimate or fraudulent.
* Visualizes transaction patterns and model performance with charts and graphs.
